Программа StudentManager прелназначается для работы студентов и администраторов учебного заведения.
В программе представлены две роли: User(Студент) и Admin.
При заходе пользователя в программу в качестве Пользователя(Студента), на его экране высвечивается приветственное сообщение.
При входе в программу в качестве Администратора(Admin), пользователь получает доступ к редактированию пользователей: Администратор имеет право удалять и добавлять пользователей.
Администратор не может удалить старшего и единственного администратора из системы(себя).
Как и администратор, так и пользователь могут вернуться к главному окну используя кнопку "Назад", которая вернет их на главный экран с Логином.
